# Changelog

All notable changes to the Cursor AI Editor Installer will be documented in this file.

## [3.0] - 2025-04-03

### Added
- Android support through Termux with comprehensive installation guide
- Status information table showing installed and latest versions
- Auto-detection of system architecture (ARM64/x86_64)
- Request ID reset feature for enhanced privacy
- Automatic dependency management

### Changed
- Improved user interface with modern styling
- Streamlined installation process
- Direct update method without requiring uninstallation
- Better error handling and user feedback

### Fixed
- Fixed symlink creation issues
- Resolved permission errors when running with sudo
- Fixed AppImage extraction on certain distributions
- Corrected configuration file handling

## [2.0] - 2025-03-15

### Added
- Command-line interface options
- Uninstall functionality
- Basic update mechanism
- Support for x86_64 architecture

### Changed
- Improved installation process
- Enhanced error reporting

## [1.0] - 2025-02-20

### Added
- Initial release
- Basic Cursor AI Editor installation
- Interactive menu interface
